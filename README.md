# 🚖 Ride Booking Analytics Dashboard (Power BI)

## 📌 Project Overview
This project analyzes ride booking data to uncover insights related to revenue trends, customer behavior, cancellations, and service quality.

The dashboard is built using Power BI and focuses on transforming raw ride data into actionable business insights through interactive visualizations and DAX-based KPIs.

---

## 🛠️ Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- CSV Dataset

---

## 📂 Dataset Description
The dataset includes ride-level information such as:

- Booking ID
- Date & Time
- Booking Value
- Booking Status (Completed / Cancelled)
- Customer ID
- Driver Ratings
- Vehicle Type
- Payment Method
- Ride Distance

Note: The dataset provided is raw data. Data transformations and calculated measures were created inside Power BI.

---

## 📊 Dashboard Pages

The report consists of **7 interactive pages**:

1. **Overall Report**
   - Total bookings, cancellations
   - Monthly booking trends
   - Booking status distribution

2. **Vehicle Report**
   - Booking value by vehicle type
   - Distance metrics (avg & total)
   - Performance comparison across vehicle categories

3. **Revenue Report**
   - Revenue by payment method
   - Revenue per KM analysis
   - Ride distance trends

4. **Cancellation Report**
   - Total cancellations and cancellation %
   - Customer vs Driver cancellation analysis
   - Reasons for cancellation

5. **Rating Report**
   - Customer vs Driver ratings
   - Vehicle-wise rating comparison

6. **Summary Report**
   - KPI overview
   - Revenue segmentation (Premium / High / Medium / Low)
   - Time-based revenue analysis

---

## 📈 Key KPIs

- **Service Quality Score**
- **Revenue Momentum**
- **Customer Retention Rate**
- **Premium Peak Hour Index**
- **Weekly Bookings**
- **Monthly Revenue**
- **Quarterly Average Booking Value**

---

## 🔍 Key Insights

- Peak hours (Morning & Evening) generate significantly higher revenue compared to off-peak periods  
- Premium ride categories contribute the highest share of total revenue  
- Customer retention plays a major role in consistent booking volume  
- Cancellation rate is around 25%, impacting overall service quality  
- UPI is the most preferred payment method  

---

## 📊 Dashboard Features

- Interactive filters (Date, Vehicle Type, Payment Mode)
- Drill-down capabilities
- KPI cards for quick insights
- Time-based trend analysis (daily, weekly, monthly)
- Clean and user-friendly UI design
